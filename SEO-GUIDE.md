# 🚀 SEO Оптимизация для Talaria.uz

## ✅ Что сделано:

### 1. Мета-теги на всех страницах:
- **index.html** — полная SEO оптимизация
- **catalog.html** — SEO для каталога
- **about.html** — SEO для страницы "О нас"
- **contacts.html** — SEO для контактов
- **product.html** — динамические мета-теги для каждого товара

### 2. Open Graph теги:
- Для корректного отображения ссылок в Telegram, WhatsApp, Facebook
- Заголовки, описания, изображения для каждой страницы

### 3. Canonical URLs:
- Указаны для всех страниц для предотвращения дублирования

### 4. Sitemap XML:
- Создан файл sitemap.xml со всеми основными страницами

### 5. Robots.txt:
- Разрешена индексация всех публичных страниц
- Запрещён доступ к папке /bot/
- Указан путь к sitemap.xml

## 📋 Следующие шаги:

### 1. Добавить сайт в Google Search Console:
1. Зайди на https://search.google.com/search-console
2. Добавь домен `talaria.uz`
3. Подтверди владение (через HTML файл или DNS запись)
4. Отправь sitemap.xml

### 2. Добавить сайт в Яндекс Вебмастер:
1. Зайди на https://webmaster.yandex.ru
2. Добавь сайт `talaria.uz`
3. Подтверди владение
4. Отправь sitemap.xml

### 3. Создать OG изображение:
Создай изображение `og-image.jpg` (1200x630px) с:
- Логотипом Talaria
- Названием "Женская обувь больших размеров в Ташкенте"
- Красивым фоном

## 📊 Как проверить SEO:

### Google:
```
https://search.google.com/search-console/about/search-analytics
```

### Яндекс:
```
https://webmaster.yandex.ru/site/talaria.uz
```

### Проверка мета-тегов:
Открой любую страницу → Правая кнопка мыши → "Просмотр кода страницы" → Найди `<meta name="description">`
